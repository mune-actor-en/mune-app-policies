# mune-app-policies

各アプリのプライバシーポリシーを GitHub Pages で配信する集約リポジトリ。

- 公開URL: https://mune-actor-en.github.io/mune-app-policies/
- 配信元: `main` ブランチの `/docs` フォルダ

## 構成

```
docs/
├── index.html                  # 全アプリのポリシー一覧
└── {app-slug}/
    └── privacy-policy.html      # 各アプリのポリシー
```
