あれいつ？ サポート

「あれっていつだったっけ？」を解決する記録アプリ

「あれいつ？」は、バイクのメンテナンスや美容室、通院、掃除など、定期的に繰り返す活動の「最後にやった日」と「次の予定」を記録・管理する iPhone 向けアプリです。経過日数の自動計算や「そろそろ」のお知らせで、やるべきタイミングを逃しません。

Qよくある質問

データはどこに保存されますか？
すべて端末内に保存され、当方のサーバーには送信されません。アカウント登録も不要です。

機種変更のときデータは移せますか？
設定のバックアップ機能から、データをファイルに保存・復元できます（写真を含めることも可能）。

購入したプレミアムを復元するには？
ペイウォール画面の「購入を復元する」から、同じ Apple ID での過去の購入を再適用できます。
